# matrix-operations
Vector and matrix implementation allowing users to perform various operations.
## Vector
* Addition
* Subtraction
* Scalar product
* Dot product
## Matrix
* Addition
* Subtraction
* Multiplication
* Square matrix text
* Scalar product
* Determinant
* Transpose
* Inverse
* Cholesky decomposition
## Built with
[**Maven**](https://maven.apache.org/) - Dependency Management
